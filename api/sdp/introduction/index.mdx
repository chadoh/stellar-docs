The Stellar Disbursement Platform (SDP) enables organizations to disburse bulk payments to recipients using Stellar.

To learn more about running the Stellar Disbursement Platform, check out the hosting guide [here](docs/sdp/host-the-sdp/). To learn more about integrating your wallet application an instance of the Stellar Disbursement Platform, checkout out our integration guide [here](/docs/sdp/integrate-the-sdp/).
